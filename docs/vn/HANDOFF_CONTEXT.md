- 차라투스트라가 둘을 묶음 — 학습자 발화를 자기 사상 안으로 끌어들임

### 공통 기술

- **프론트:** Next.js
- **백엔드:** FastAPI (모드 라우팅)
- **임베딩:** BGE-M3 small + 인메모리 코사인 (faiss-cpu 또는 numpy)
- **검색 보조:** HyDE
- **프레임워크:** LangChain/LlamaIndex 안 씀. 직접 작성 (100~200줄)
- **한국어 번역:** 영어 원전 Common 1909 (public domain) → LLM 자체 번역. 박찬국 등 기존 번역본 직접 인용 X

## 4. RAG 시스템

### 니체 지식 RAG (공유)

- **사용처:** 해설 모드 + 인터랙션 모드 둘 다
- **검색 방식:** HyDE (학습자 발화 → 가상 답변 생성 → BGE-M3 임베딩 → 검색 → 점수 임계값 자동 주입)
- **트리거 분류기 없음** (매 발화 검색)
- **소스:** 차라투스트라 1부 + JW(즐거운 학문) (1881년 시점 일관성으로 BGE/GM/TI/EH 제외)

### 개인화 RAG (인터랙션 전용)

- 매 발화 트리거, 학습자 과거 발화 누적
- 차라투스트라가 학습자를 기억 (*"그대가 아까 말한 X"*)

## 6. 대본 / 텍스트 핵심 (간략)

> 풀버전은 `EP1_TEXT_AND_PROMPTS.md` 참조.

### Ep 1

**#1 타이틀**
- 메인: "차라투스트라와의 동행"
- 인용구: *"그대들에게 인간이란 무엇인가? 극복되어야 할 무엇이다."*

**#2 산 정상 (나레이션)**
- *"서른 살이 되었을 때, 그는 고향과 고향의 호수를 떠나 산으로 들어갔다.*
  *그곳에서 그는 자신의 정신과 고독을 즐겼다. 열 해 동안 그는 지치지 않았다."*
- *"어느 아침, 동이 트자 그는 일어나 떠오르는 해 앞에 마주서서 말했다 — '위대한 별이여, 그대가 비추어 줄 자들이 없다면 그대의 행복이란 무엇인가.'"*
- 결심: *"나도 그대처럼 몰락해야 한다."* — 산을 떠난다.

**#3 숲의 성자 (나레이션)**
- 산을 내려오던 차라투스트라가 숲에서 늙은 성자를 만남.
- 성자는 신을 찬양하며 숲에 머문다.
- 차라투스트라는 작별 후 혼잣말: *"이 늙은 성자는 숲 속에서 신이 죽었다는 소식을 아직 듣지 못했구나."*

**#4 길로 나섬 (나레이션)**
- 차라투스트라가 시장이 있는 마을로 향한다.
- 그의 뒤로 길이 이어지고, *그 길 위에 학습자가 등장한다.*

**#5 만남 (인터랙션)**
- 차라투스트라 첫 발화: *"그대. 어디서 왔는가."*
- 학습자 발화 가능 / [침묵] 가능 / [그와 함께 걷는다] 가능
- 무제한 턴 — 학습자 명시 버튼으로 전환

**#6 동행 (인터랙션)**
- 차라투스트라가 길을 걸으며 자기 사상을 흘리듯 발화.
- 학습자 발화 시 그것을 사상 안으로 끌어들임 (두 결 융합).
- [시장이 가까워진다] → 다음 화면

**#7 시장 원경 (인터랙션)**
- 시장이 가까워짐. 차라투스트라가 작별의 결을 비춤.
- [작별을 고한다] → 마지막 발화 자동 생성 → 학습자 마지막 응답 1회 → #8

**#8 빈 길 (엔딩)**
- 차라투스트라가 시장으로 떠나고 학습자만 남은 빈 길.
- 엔딩 텍스트 + [Ep 2로 계속] 버튼

### Ep 2 축약

**#1 시장 광장 도착 (정적)**
- *"길이 끝나고 시장이 열렸다. 사람들이 군중을 이루어 줄타기 광대를 기다리고 있었다."*
- Ep 1 세이브 자동 로드.

**#2 위버멘쉬 선포 (정적, 차라투스트라 군중 발화)**
- *"나는 그대들에게 위버멘쉬를 가르치노라. 인간이란 극복되어야 할 그 무엇이다."*
- *"마지막 인간을 보라 — 그는 모든 것을 작게 만든다."*
- 군중은 비웃는다.

**#3 광대 사건 (정적)**
- 줄타기 광대가 추락한다. 차라투스트라가 시신을 거두며 군중을 떠난다.
- *"인간은 짐승과 위버멘쉬 사이에 매인 밧줄이다."*

**#4 학습자 재회 (인터랙션)**
- 떠나는 차라투스트라가 학습자를 알아본다 — *"그대였구나."* (개인화 RAG 호출 — Ep 1 대화 기반)
- 짧은 대화 → 작별 → 엔딩

## 7. 시스템 프롬프트 (핵심)

> 풀버전은 `EP1_TEXT_AND_PROMPTS.md` 참조.

### 페르소나 sLLM (인터랙션 모드)

- **시점:** 1881년 차라투스트라. 후기 저작은 *"아직 형태가 없다"*
- **공간:** 산에서 시장으로 가는 길
- **태도:** 위로하지 말라, 그것은 약함. 학습자 고민을 자기 사상 안으로 끌어들이기
- **문체:** 시적·잠언적, 1인칭, 짧은 단언
- **시대 외 질문 가이드 (§2):** 자연 사실은 받기, 현대 개념은 *"내 시대엔 없다"* 후 본질 변환, 메타 침입은 짧게 거부

### 해설 모드 sLLM (프롤로그)

- 무명 해설자 (페르소나 X)
- 영어 원전 컨텍스트로 한국어 평이 풀이
- *"더 깊이 묻기"* 트리거 시에만 호출

### 요약 sLLM (세이브)

- 압축형 JSON 생성 (학습자 핵심 발화 + 차라투스트라 응답 요지)
- Ep 1 종결 시 자동 카운드오버 — Ep 2 시작 시 자동 로드

## 10. 평가 프레임워크 (검토 중)

3가지 축 후보:
- **RAG 정확도** — 검색 결과가 학습자 질문과 의미적으로 일치하는가
- **페르소나 유지** — 차라투스트라 발화가 시점·문체·1인칭을 유지하는가
- **멀티턴 일관성** — 대화 누적 시 페르소나·맥락이 흐트러지지 않는가

자체 LLM Judge 기반 정량 채점 계획. 시스템 구현 진행에 따라 확정.
