## 0. 사전 준비 (시연 30분 전)

### 환경 셋업

전제: llama-server 2개(chat :8000 + embed :8001)가 떠 있어야 해설 모드 RAG가 동작.
자세한 llama-server 기동·모델 파일 위치는 `app/README.md` "전제" 절 참조.

```powershell
# 통합 가동 (ml-backend :3001 + frontend :3000 + Electron 동시)
cd app
npm install   # 최초 1회
npm run dev
```

### 모드 확인
- 해설 모드: `ml-backend`의 llama.cpp 기반 RAG 라이브. `http://localhost:3001/health`에 `indexed_chunks` 응답.
- 인터랙션 페르소나 / 요약 sLLM: 미연결 (Phase 9 잔여). 시연은 자동 발화 + 학습자 발화 흐름의 *시각 시퀀스*만.

### 시연 전 1회 워밍업
- `/` 타이틀 → [시작] → /ep1/scene/2 → ▼ 인디케이터 보이는지 확인
- /ep1/scene/5 → 자동 발화 → [발화하기] 1회 → 응답 스트리밍 확인
- /ep1/ending → [Ep 2로 계속] → transition 동작 확인
- /ep2/scene/4 → 자동 발화 + 작별 흐름까지

## 2. 라이브 시연 시나리오

### 화면 #1 타이틀 (~10초)

- **시연자 행동**: 타이틀 화면에서 잠시 머무름 (인용구 보여줌)
- **음성 가이드**: *"차라투스트라와의 동행. 니체의 『차라투스트라는 이렇게 말했다』를 학습자가 직접 따라가는 비주얼 노벨입니다."*
- 클릭: **[ 시 작 ]**

### 화면 #2 산 정상 (~30초, Ep 1 정적 나레이션)

- **시연자 행동**: 단락 5개를 천천히 클릭하며 진행
- **음성 가이드** (단락 1 즈음): *"서른 살이 되었을 때, 그는 산으로 들어갔다."*
- 핵심 단락: **#4 차라투스트라 첫 발화**(인용 italic)와 **#5 강조** *"산을 내려가야 한다."*
- 1회 — **[해설] 클릭** → 우측 패널 슬라이드 → 정적 풀이 보여줌 (1회만 시연, 패널 닫고 진행)
- 마지막 단락 클릭 → /ep1/scene/3

### 화면 #3 숲의 성자 (~25초)

- 단락 6개 클릭
- 핵심: 차라투스트라 **"나는 인간을 사랑한다"** (강조 직립체)
- 다음 화면

### 화면 #4 길로 나섬 (~20초, **분위기 전환점**)

- 단락 3개 — 짧지만 무게감
- 핵심: **"신이 죽었다는 소식"** (강조)
- 마지막 단락 클릭 → **#5로 진입 시 800ms 슬로우 페이드** 강조

### 화면 #5 만남 (~50초, 첫 인터랙션)

- 진입 시 자동 발화: **"그대. 어디서 왔는가."** (시뮬 스트리밍 80ms/char)
- **시연자 발화 예시**: *"산에서 내려오는 길에 그대를 만났습니다. 어디로 가십니까."*
- 차라투스트라 응답 mock 스트리밍 (~5초)
- 한 턴 더: **[침묵]** 클릭 — 학습자도 침묵을 *행위*로 선택할 수 있음 시연
- 차라투스트라가 침묵 응답 mock
- **[그와 함께 걷는다 →]** 클릭

### 화면 #6 동행 (~40초)

- 자동 발화 (sLLM auto)
- **시연자 발화**: *"신이 죽었다는 말씀이 무슨 뜻입니까."* (작품 핵심 사상 질문)
- 응답 — 풀이 RAG는 mock이라 단순. 실제 vLLM 환경에서 *작품 결과 풀이*가 자연스럽게 녹아듦을 설명.
- **[시장이 가까워진다 →]**

### 화면 #7 시장 원경 (~50초, **작별 시퀀스**)

- 자동 발화 (작별 결 비춤)

### 화면 #8 빈 길 (Ep 1 엔딩, ~15초)

- 일러스트 5초 정적 → 텍스트 페이드인 *"시간은 흐른다. 학습자가 무엇을 하든."* → 메뉴

## 6. Q&A 예상 질문 + 답변 결

1. **"실제로 sLLM이 답하는가, mock인가?"** — 해설 모드는 라이브 RAG (llama.cpp 기반, 2026-05-16 통합). 인터랙션 페르소나·요약은 Phase 9 잔여라 현재 시연은 시각 시퀀스 중심.
2. **"학습자가 한 발화를 차라투스트라가 *기억*하는가?"** — 화면 안 turn 누적은 컨텍스트로 들어감. Ep 1 → Ep 2는 카운드오버 요약 sLLM이 다리. 현재 시연은 시각만.
3. **"왜 책 삽화 레이아웃인가?"** — 작품의 *읽는 결*을 화면에 옮기기 위함. 챗봇 결로 가면 작품 무게감이 깨짐.
4. **"강조 표시 양식이 책 강조와 같은가?"** — 진한 잉크 + 굵게 + quote 안 직립체로 분리. 책의 강조 양식 그대로.
5. **"Ep 3는 왜 슬라이드인가?"** — 본 캡스톤은 Ep 1+2로 *가능성 입증*에 한정. Ep 3~22는 같은 패턴 반복이라 확장 비전으로 분리.
